# 云秒搭AI周报 - 像素风重构版

## 🎮 新特性
- **像素风UI设计** - 复古游戏风格界面
- **纯静态架构** - 完美兼容EdgeOne Pages
- **独立文章页面** - 每篇文章都有独立HTML文件
- **响应式设计** - 支持移动端和桌面端

## 🚀 部署说明

### EdgeOne Pages部署
1. **构建命令**: `python3 rebuild_site.py`
2. **输出目录**: `/` (根目录)
3. **分支**: `main`

### 文件结构
```
/
├── index.html          # 像素风主页
├── styles.css          # 像素风样式
├── articles/           # 独立文章页面
│   ├── article1.html
│   ├── article2.html
│   └── ...
├── posts/              # 文章数据
│   └── articles.json
└── images/             # 图片资源
    └── ...
```
